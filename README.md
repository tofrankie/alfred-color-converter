# alfred-color-converter

[![npm version](https://img.shields.io/npm/v/alfred-color-converter)](https://www.npmjs.com/package/alfred-color-converter) [![node version](https://img.shields.io/node/v/alfred-color-converter)](https://nodejs.org) [![npm package license](https://img.shields.io/npm/l/alfred-color-converter)](https://github.com/tofrankie/example/blob/main/LICENSE) [![npm last update](https://img.shields.io/npm/last-update/alfred-color-converter)](https://www.npmjs.com/package/alfred-color-converter)

💻 An Alfred workflow for color conversion.

- Convert between hex/rgba colors and decimal color values.
- Convert [CSS named colors](https://drafts.csswg.org/css-color/#named-colors) to hex/rgba or decimal values.

## Installation

> [!NOTE]
>
> - Requires Node.js 12 or later.
> - Requires zsh ([Apple migrated the default shell to zsh](https://support.apple.com/en-us/HT208050)).

```bash
# Install globally
$ npm i alfred-color-converter -g

# Import to Alfred
$ alfred-color-converter-import
```

## Usage

In Alfred, type `cc`, then press <kbd>Enter</kbd> to copy the result to the clipboard.
